== Installation ==

1. Unzip files.
2. Upload "woocommerce-jne" folder into your plugins directory.
3. Activate the plugin.
4. Go to WooCommerce > Settings > Shipping > JNE Shipping.
5. Configure settings to your needs.
6. Have fun!
